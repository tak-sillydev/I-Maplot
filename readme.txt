JMAEQ I-Maplot v1.2.0
気象庁防災情報XMLフォーマット形式電文を30秒おきに取得解析し、新たな地震情報がある場合に震度地図を描画する。


本システムで使用している地図:
 - 「国土数値情報（湖沼データ）」（国土交通省）（https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-W09-2005.html）を加工して作成
 - 日本地図（細分区域図）は、「地震情報／細分区域」（気象庁）を以下のように加工して使用しています。
   - 加工の手順：
　   1. シェープファイルを「mapshaper」（https://mapshaper.org/）で開く。
　   2. 「prevent shape removal」「Visvalingam/weighted area」にそれぞれチェックの上、0.1%でsimplify。
　   3. 上記チェックはあってもなくても多分さほど結果に影響しない。
　   4. Self-intersection を削除しエクスポート。
